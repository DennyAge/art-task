Artworks List Application
-

**Technologies:**
Next.js

Setup and Running
-
**Clone the repository:**
```bash
git clone https://github.com/DennyAge/artworks-list
```
```bash
cd artworks-list
```

**Install dependencies:**

```bash
npm install
```
**Run app:**
```bash
npm run dev
```

App will be available at: http://localhost:3000.
